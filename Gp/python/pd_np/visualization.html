Matplotlib¶

  • color : 선 색상
  • linestyle : 선 스타일
  • linewidth : 굵기
  • marker : 마커 설정 ---- 'solid', 'dashed', 'dashdot', 'dotted'
  • label : 범례 설정
  • drawstyle 연결되어 있는 선 연결하기 위한 스타일 지정
  • ax : 서브 플롯 지정
  • edgecolors :테두리색 --- 'face', 'none'
